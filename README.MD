dohko-springcloud-example
---
基于SpringCloud的Example，逐步更新完善

涉及功能组件：

1. 注册中心

    1）Eureka
    
    2）Nacos
    
    3）Consul 
2. 网关

    1）Zuul
    
    2）Springcloud Gateway

3. 配置中心
    
    1）Apollo
    
    2）Nacos
    
4. 服务间调用
    
    1）Feign
    
    2）MQ
 
 5. 熔断限流
 
    1）Hystrix
    
    2）Sentinal
    
 6. 分布式事务
